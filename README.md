## ✨ Features

- **Multi-Instance Support**: Automatically detects and injects payloads into all running instances of the game simultaneously, including those running in Sandboxie environments.
- **God Mode**: Locks Health (HP) at absolute maximum (`99.9M`), making your hero completely invulnerable.
- **Insta Kill**: Locks Attack Damage at maximum (`99.9M`) for guaranteed One-Hit Kills against any monster.
- **Adjustable Speed Hack**: Supercharge your Attack Speed and Movement Speed. Includes precision sliders to prevent server de-syncs and stage reset loops.
- **Auto-Save Configuration**: Seamlessly remembers your toggle states and slider values via `farm_config.json`.
- **Premium UI**: Built with `CustomTkinter` and `pywinstyles` for a gorgeous dark-neon glassmorphism aesthetic.

## 🛠️ Installation

This project has been compiled into a standalone, dependency-free Windows executable. No Python installation is required for end users.

1. **Download** the latest release package.
2. Extract the folder to your desired location.
3. That's it! Keep `Phantom_Client.exe` and any generated config files in the same directory.

*(For developers: Requirements are listed in `requirements.txt`. Built using PyInstaller).*

---

## 🚀 Usage

1. **Launch the Game**: Start your instance(s) of *Task Bar Hero* and log in.
2. **Run the Injector**: Double-click `Phantom_Client.exe`.
3. **Inject**: Click the **INJETAR** (Inject) button on the left sidebar. The built-in console will display real-time memory connection statuses.
4. **Tweak & Farm**: Toggle your desired hacks and adjust the speed sliders on the fly.

> [!WARNING]  
> **Stage Reset Bug**: If the game frantically resets to stage 1 without progressing, your hero is moving too fast for the game engine to spawn items. **Fix**: Slightly reduce the *Movement Speed* slider.
